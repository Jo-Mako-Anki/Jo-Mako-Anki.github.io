Reading Course

Information

One of the most frustrating things when learning a new language is that it's very hard to find material suited to your knowledge.
This is where this course comes in.


You will be given a list of vocabulary, a list of grammar, and then the text to read.
I'll go over the text as well giving explanation about what's going on.


The goal with this course is to focus on reading, so I'll make sure that you only have a few new words between each new material.
I'll also only use material that will be available to you free.
I picked 3 sources:

  • openings from jrpg
  • manga chapters from bilingualmanga.com
  • short fairy tales and stories from http://hukumusume.com.

Tips

Use online dictionary

All reading can be done online, so you can use Rikaikun, Yomichan or 10ten reader (Chrome / Mozilla).

Hover over Japanese text in the browser to have an instant dictionary pop-up.
